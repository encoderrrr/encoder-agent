# Demo Script

Use this script for a short live demo or screen recording.

## Demo Goal

Show that `@encoderagent` can:

- receive a DM
- quote a service
- request payment
- get paid
- complete the job

## Live Demo Script

1. Show the live agent nametag:
`@encoderagent`

2. Send:

```text
catalog
```

3. Show the service list.

4. Send:

```text
quote summarize | Unicity is building infrastructure for autonomous AI agents that can transact and settle automatically.
```

5. Show the returned `jobId` and quote.

6. Send:

```text
accept <jobId>
```

7. Show the payment request.

8. Pay the request.

9. Send:

```text
status <jobId>
```

10. Show that the job is `completed`.

## Good Screens To Capture

- `catalog` output
- quote output
- payment request output
- final completed status

## One-Line Pitch

`A DM-native paid service agent on Unicity testnet that quotes work, requests payment, and automatically delivers the result back to the user.`
